Follow these steps to successfully add this report to your workspace:

1. Log in to your local CxReport instance
2. Go to Workspace Configuration > Data Import and import the CxReportsExport.json
3. Find the new report at the Reporting > Reports page

For additional information please visit https://docs.cx-reports.com/
If you require assistance or you would like to report a problem, please use the "Report a problem" option on the website or within the application.
